## 📥 Installation

```bash
# Clone repo
git clone https://github.com/samir20-23/your-chatbot-repo.git
cd your-chatbot-repo

# Setup backend
cd server
npm install
cp .env.example .env  # add your API keys

# Setup frontend
cd ../client
npm install
```

## ⚙️ Run Locally

```bash
# In /server
npm start
# In /client
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to chat with the bot.
